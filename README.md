# Claude Architect — Enterprise Program

An intensive 3-day program for engineers and architects building production-grade AI systems with Claude. Covers Claude ecosystem foundations, enterprise governance, Claude Code, multi-agent architecture, CI/CD integration, and context engineering.

## Program Structure

| Day | Track | Modules |
|-----|-------|---------|
| Day 1 | Ecosystem & Foundations | Claude Ecosystem & Enterprise Foundations · Claude Cowork · Enterprise Governance & Responsible AI · Labs |
| Day 2 | Claude Code & Development | Claude Code Practitioner · AI-Assisted Dev Workflows · Prompt Engineering for Code · Developer Productivity Automation · Labs |
| Day 3 | Architecture | Context Engineering & CLAUDE.md · CI/CD & GitHub Integration · Multi-Agent Architecture · Enterprise AI Platform Architecture · Labs |

## Quick Start

No build step — open `index.html` directly in any modern browser.

```bash
open index.html
# or serve locally:
npx serve .
```

## Features

- **Persistent sidebar** — navigate to any module or lab from any page; all day sections stay expanded
- **Live tweaks panel** — adjust accent color, heading font, density, and author info in real time (settings persisted to `localStorage`)
- **TOC scroll spy** — "On This Page" sidebar section auto-highlights current section
- **Code copy buttons** — one-click copy on all code blocks
- **Responsive** — mobile-friendly with hamburger sidebar toggle
